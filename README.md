**[Target‐specific precision of CRISPR‐mediated genome editing]()** (2018)

*Chakrabarti AM, *Henser-Brownhill T, *Monserrat J, Poetsch AR, Luscombe NM, Scaffidi P

Raw data are deposited at EBI ArrayExpress:

* E-MTAB-7091
* E-MTAB-7095
